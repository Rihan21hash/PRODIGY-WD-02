## Live Demo

**Website:** https://rihan21hash.github.io/PRODIGY-WD-02/

## About The Project

The Stopwatch Web Application is designed to provide precise time measurement with essential stopwatch functionalities. Users can start, pause, reset, and record lap times while monitoring cumulative timing information in real time.

The project focuses on delivering a clean user experience through a modern dark-themed interface, responsive layout, and keyboard shortcut support for improved accessibility and usability.

## Features

- Start and pause stopwatch functionality
- Reset stopwatch instantly
- Record multiple lap times
- Display cumulative lap information
- Real-time stopwatch updates
- Keyboard shortcut support
- Responsive design for all devices
- Modern and minimal user interface
